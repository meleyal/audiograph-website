FAQ

What audio formats do you support?

We support WAV, MP3, FLAC, and AAC formats. Maximum file size is 500MB. All responses are returned in the same format as the input file unless specified otherwise.

How is audio processing time calculated?

Processing time is based on the duration of the input audio file, rounded up to the nearest minute. For example, a 2:30 minute file will be charged as 3 minutes.

What happens to my audio files after processing?

Files are automatically deleted after processing. We never store audio content unless explicitly requested. All processing is done in memory and follows strict privacy guidelines.

What's your uptime guarantee?

We maintain 99% uptime, backed by a financially guaranteed SLA for enterprise customers. Our infrastructure is distributed across multiple regions for redundancy.

What are your rate limits?

Standard accounts can process up to 60 concurrent requests. Enterprise accounts receive higher limits and can request custom configurations based on their needs.

How do I get notified when processing is complete?

You can either provide a webhook URL to receive callbacks, or use long polling to check the status. Both methods notify you of completion or failure, with automatic retries configurable.
